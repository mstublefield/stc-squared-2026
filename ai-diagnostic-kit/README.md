# Team Diagnostic AI Kit

## What This Is

An AI-powered diagnostic tool that helps you identify invisible patterns killing your team's productivity. Based on the 5-lens diagnostic framework by Matthew Stublefield.

Takes about 5 minutes for a quick diagnostic. Go deeper on any lens if you want.

## Quick Start (Any AI Tool)

The fastest way to get started – no setup required:

1. Open ChatGPT, Claude, Gemini, or any AI chat tool
2. Copy the entire contents of [`instructions.md`](instructions.md) and paste it as your first message
3. Send it, then follow the AI's prompts

This works well for a quick session. For a richer experience, use one of the setup options below – they give the AI access to the full diagnostic framework so it can provide deeper, more specific insights.

## Setup by Platform

### ChatGPT (Custom GPT)

1. Go to **Explore GPTs** > **Create**
2. In the **Instructions** field, paste the full contents of `instructions.md`
3. Under **Knowledge**, upload `diagnostic-framework.md`
4. Name it something like "Team Diagnostic"
5. Set to **Only me** (or share with your team if you want)
6. Save and start a conversation

### Claude (Project)

1. Create a new **Project**
2. In the **Project Instructions**, paste the full contents of `instructions.md`
3. Add `diagnostic-framework.md` as a project knowledge file
4. Start a new conversation within the project

### Google Gemini (Gem)

1. Go to **Gem manager** > **New Gem**
2. In the **Instructions** field, paste the full contents of `instructions.md`
3. Upload `diagnostic-framework.md` as a knowledge file
4. Save and start chatting

### Other AI Tools

Most AI tools support custom instructions and file uploads. The pattern is the same:
- `instructions.md` goes into the system prompt or custom instructions
- `diagnostic-framework.md` gets uploaded as reference material or context

If your tool only supports a single input, paste both files into the conversation – instructions first, then the framework – and tell the AI to use them.

## What to Expect

**Phase 1 (~5 minutes):** Five quick multiple-choice questions, one per diagnostic lens. You'll get a snapshot of where your team's biggest constraints likely are.

**Phase 2 (optional):** Pick any lens to go deeper. The AI will ask more specific questions and help you identify patterns – like having a diagnostic coach walk you through it.
